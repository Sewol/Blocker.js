Blocker.js

Draw blocks on webpages


Click and Drag anywhere on this page, while holding w, g or b to draw blocks. To draw on other websites, drag and drop this link Blocker to your favourites bar.


Blocker is a javascript function that allows the user to click and drag to draw opaque rectangles on the screen. To run on another page, just click the script in the favourites bar.

It's a small personal project that may have a few uses for anyone who wants to use it.

Holding w, g or b as you draw will draw the block in white, grey and black respectively. Currently, there is no way to erase single blocks. Refreshing the page will clear them all.

Blocker has been tested in the following browsers:

  • Mozilla Firefox - Works fine
  • Google Chrome - Works great
  • Safari - Works fine
  • Microsoft IE - Works fine in IE9
  • Opera - Works great

The source is available here, although a site will be needed to host it if you make any changes to it.
